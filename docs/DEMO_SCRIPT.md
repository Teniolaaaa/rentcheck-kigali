# DEMO SCRIPT for RentCheck Kigali

## Introduction
Welcome to the demo of RentCheck Kigali, a web application designed to help users find reliable rental properties in Kigali. In this demo, we will showcase the key features of the application, including searching for landlords, viewing property details, sorting and filtering results, and understanding the RentCheck Score.

## Step 1: Searching for a Landlord
- Open the RentCheck Kigali application in your web browser.
- Locate the search bar at the top of the page.
- Type in the name of a landlord or the address of a property you are interested in.
- Press "Enter" or click the search icon to initiate the search.

## Step 2: Viewing Results
- After the search is executed, the results will be displayed in a card format below the search bar.
- Each card will show essential information about the property, including the RentCheck Score, address, and any relevant complaints or disputes.

## Step 3: Sorting and Filtering
- To sort the results, use the sorting controls available on the page.
- You can sort by:
  - Number of complaints
  - Risk level
  - Dispute count
- To filter the results, open the filter panel.
- You can filter by:
  - District (Gasabo, Kicukiro, Nyarugenge)
  - Verified complaints
  - Scam flags
- Apply the desired filters and observe how the results update in real-time.

## Step 4: Understanding the RentCheck Score
- Click on a property card to view more details.
- The property details page will include a breakdown of the RentCheck Score.
- The score is calculated based on:
  - Legal/dispute history (35%)
  - Scam or fraud reports (25%)
  - Tenant reviews and verified complaints (25%)
  - Trend over time (15%)
- Review the explanation provided to understand how the score reflects the property's reliability.

## Step 5: Error Handling
- If you enter an invalid landlord name or address, the application will display a user-friendly error message.
- The application gracefully handles API failures, ensuring that users are informed without crashing the interface.
